# SwingPick
Daily swing trade suggestions based on technical indicators.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Copy `.env.example` to `.env` and add your Alpaca API keys:
```bash
cp .env.example .env
```

3. Get free API keys at https://alpaca.markets/

4. Run the app:
```bash
streamlit run app.py
```
